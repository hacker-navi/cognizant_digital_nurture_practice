# Core Java Exercises

This folder contains starter solutions for the 41 exercises you listed.

## Notes

- Most examples are self-contained and use only the Java standard library.
- The JDBC examples use SQLite-style URLs as a simple local option, but you still need a JDBC driver on the classpath to run them.
- The module, record, pattern-matching switch, and virtual-thread examples assume Java 21 unless otherwise noted.
- `output.txt` is included as a sample file for the file-reading exercise.
